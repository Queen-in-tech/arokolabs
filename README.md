# Àrokò Labs

AI Automation & Intelligent Systems agency website.

Built with **Next.js 16 App Router** + **Tailwind CSS v4** + **Framer Motion** + **TypeScript**.

## Stack
- Next.js 16 (App Router, fully static)
- Tailwind CSS v4
- Framer Motion
- TypeScript
- Deployed on Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Push to `main` — Vercel auto-deploys.
